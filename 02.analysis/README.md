
## 260205
This directory contains example notebook to analyze and reproduce figures in our manuscript.

`01.clustering`: clustering of RNA and DNA data using Seurat and SnapATAC2<br>
`02.integration`: CCA integration between RNA and with snATAC<br>
`03.chromhmm`: run chromhmm to determine chromatin states with four modalities<br>
`04.ccre`: classification and characterization of cCREs using chromatin state annotation<br>
`05.abc`: predict enhancer-promoter interactions with abc model<br>
`06.grn`: use SCENIC+ to reconstruct active and repressive gene regulatory networks<br>
`07.gwas_variants`: disease risk variants enrichment analysis and effect prediction<br>
`08.heterogenity`: structual and regional heterogentiy in RNA expression and epigenome<br>
`09.spatial`: MERFISH analysis and spatial information imputation<br>
`10.hic`: annotate short-range and long-range loops called from snm3C-seq<br>
`11.conservation`: identify convergent and diveregent TF programs
