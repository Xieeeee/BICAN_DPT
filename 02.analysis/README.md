
## 260205
This directory contains example notebook to analyze and reproduce figures in our manuscript.

`01.clustering`: clustering of RNA and DNA data using Seurat and SnapATAC2
`02.integration`: CCA integration between RNA and with snATAC
`03.chromhmm`: run chromhmm to determine chromatin states with four modalities
`04.ccre`: classification and characterization of cCREs using chromatin state annotation
`05.abc`: predict enhancer-promoter interactions with abc model
`06.grn`: use SCENIC+ to reconstruct active and repressive gene regulatory networks
`07.gwas_variants`: disease risk variants enrichment analysis and effect prediction
`08.heterogenity`: structual and regional heterogentiy in RNA expression and epigenome
`09.spatial`: MERFISH analysis and spatial information imputation
`10.hic`: annotate short-range and long-range loops called from snm3C-seq
`11.conservation`: identify convergent and diveregent TF programs
